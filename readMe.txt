Lancer l'application web en locale :
        - à la racine du projet faire la commande "php -S localhost:8000"
        - Se rendre sur votre navigateur web à l'adresse "localhost:8000"

Technologies utilisées :
    - HTML/CSS
    - PHP
    - SQLITE3 (avec son logiciel d'interface DB BROWSER)
    - Bootstrap

L'application permet à l'utilisateur de créer un compte sur le site web, afin d'avoir accès à 
son espace membre où il pourra consulter l'ensemble de ses cartes de visites ainsi qu'en ajouter des 
nouvelles au besoin.
